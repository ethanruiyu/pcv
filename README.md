# The Point Cloud Web Viewer

## Overview

+ Support common point cloud formats(**LAS**, **DRACO**, **PLY**, **OBJ**)
+ Based on [Deck.gl](https://github.com/visgl/deck.gl) High-performance rendering

## Installation

### Development
+ `$ git clone github.com/ethanruiyu/pcv.git`
+ `$ yarn`
+ `$ yarn start`

### Static Build
+ `$ git clone github.com/ethanruiyu/pcv.git`
+ `$ yarn`
+ `$ yarn build`
